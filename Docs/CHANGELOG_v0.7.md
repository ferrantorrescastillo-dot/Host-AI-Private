# CHANGELOG v0.7 - HOST AI

## Objetivo

Mejorar el uso diario de Host AI en cocina.

La v0.7 empieza a tratar la producción disponible como una pieza central del sistema.

---

## Cambios principales

### Dashboard Cocina mejorado

Ahora muestra:

- primera tarea recomendada
- próximos eventos
- producción pendiente
- producción disponible
- avisos simples de caducidad

---

### Producción Disponible

Se añade la posibilidad de registrar consumos de producción disponible.

Ejemplo:

- Hay 38 L de fumet.
- Se usan 8 L en servicio.
- Host AI deja registrados 30 L.

---

### Rendimientos

El Dashboard Rentabilidad ya muestra rendimientos reales cuando existen datos.

Ejemplo:

- previsto medio
- real medio
- diferencia media

---

## Filosofía v0.7

Host AI empieza a trabajar con la lógica real de cocina:

- producir
- guardar
- consumir
- revisar lo disponible
- aprender rendimientos reales

---

## Próximo paso

v0.8 debería centrarse en compras inteligentes:

Evento → Producción → Ingredientes → Stock → Pedido por proveedor
