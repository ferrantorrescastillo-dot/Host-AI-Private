# Host AI 2.1 Sprint 4.5

## Compras / Pedidos

Este sprint conecta stock con compras.

### Nuevo

- `SERVICIOS/compras_servicio.py`
- `MODULOS/compras.py`
- Opción `15. Compras / Pedidos`

### Funciones

1. Ver productos bajo stock mínimo.
2. Generar pedido por stock mínimo.
3. Ver pedidos generados.
4. Ver último pedido.
5. Exportar pedido a TXT.
6. Marcar pedido como enviado.

### Nota

Este sprint aún no genera compras desde recetas/eventos reales. Eso llegará cuando el Motor Gastronómico convierta platos en elaboraciones e ingredientes.
