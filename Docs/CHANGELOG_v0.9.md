# CHANGELOG v0.9

## Motor de Planificación Inteligente

- Añadido horario inteligente por persona.
- Host AI ordena tareas por prioridad, duración y tipo de elaboración.
- Se guardan hora de inicio estimada, hora de fin estimada y nota de planificación.
- Añadida opción para ver horario guardado.
- La planificación empieza a comportarse como un jefe de producción: tareas largas y fondos primero, tareas cortas en huecos y tareas finales cerca del servicio.

## Objetivo

Pasar de una lista de tareas a un primer planning operativo de cocina.
